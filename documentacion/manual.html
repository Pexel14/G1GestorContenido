Manual de Usuario

1. Introducción

Esta aplicación permite añadir, modificar y eliminar desafíos y experiencias para la aplicación del proyecto integrador.

2. Instalación y Configuración

Para ejecutar la aplicación compila el código en NetBeans.

3. Uso Básico de la Aplicación

  • Ejecutar la aplicación.
  • Al ejecutar tienes la opción de moverte entre pantallas pulsando las tres primeras imágenes de la izquierda.

    • Ventana Añadir

  • Si la primera imagen es pulsada, se cargará la ventana 'Añadir'.
    • En la ventana 'Añadir', se puede rellenar los campos necesarios para crear un desafío.
    • Después de rellenar los campos correctamente y pulsar el botón 'Añadir Desafío', los campos se desactivarán y aparecerán los campos para añadir las experiencas.
    • Al pulsar el botón 'Añadir Experiencia'se añadirá la experiencia. Se pueden rellenar tantas experiencas como se quieran.
    • Cuando se haya terminado de añadir experiencias, al pulsar el botón 'Crear Desafío' se subirá el desafío a la aplicación.

      • Vista de la ventana principal Añadir:

    Ventana Modificar

  • Si la segunda imagen es pulsada, se cargará la ventana 'Modificar'.
    • En la ventana 'Modificar', se tiene que introducir el título del desafío.
    • Al pulsar el botón 'Buscar', se rellenarán los campos modificables de un desafío y de una experiencia.
    • Si el desafío tiene varias experiencias, se podrá rotar entre ellas con los botones situados a los lados de los campos de la experiencia.
    • Para realizar la modificación de un desafío, se debe pulsar el botón 'Modificar Desafío'
    • Para realizar la modificación de una experiencia, se debe pulsar el botón 'Modificar Experiencia'

      • Vista de la ventana Modificar:

    Ventana Eliminar

  • Si la tercera imagen es pulsada, se cargará la ventana 'Eliminar'.
    • En la ventana 'Eliminar', se tiene que introducir el título del desafío.
    • Al pulsar el botón 'Buscar', se rellenarán los campos con una experiencia del desafío.
    • Si el desafío tiene varias experiencias, se podrá rotar entre ellas con los botones situados a los lados de los campos de la experiencia.
    • Para eliminar una experiencia, pulsa el botón 'Eliminar Experiencia'.
    • Para eliminar una desafío, pulsa el botón 'Eliminar Desafío', se eliminarán las experiencias relacionadas al desafío.

      • Vista de la ventana Eliminar:

    Salir

  • Para salir de la aplicación, pulsa la cuarta imagen.
  • Para consultar el manual de usuario pulse la imagen situada arriba a la derecha.
